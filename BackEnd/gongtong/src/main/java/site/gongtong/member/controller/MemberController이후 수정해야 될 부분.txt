민감한 작업은 주로 POST 요청이 아닌, HTTPS를 통한 POST 요청 또는 다른 안전한 방법으로 처리하는 것이 권장됩니다
HTTPS를 사용하려면 SSL/TLS 인증서를 구입하고 서버에 설치하여 설정해야 합니다
=>
1. SSL/TLS 인증서 획득:

   SSL/TLS 인증서를 구입하거나 Let's Encrypt와 같은 무료 인증 기관에서 인증서를 발급받습니다.

2. Spring Boot 애플리케이션 설정:

   application.properties 또는 application.yml 파일에 HTTPS 설정을 추가합니다.

   ```
   server:
     port: 8443
     ssl:
       key-store: classpath:keystore.jks
       key-store-password: your-keystore-password
       key-store-type: PKCS12
       key-alias: tomcat
   ```

1. 인증서 파일 준비:

  프로젝트에 keystore 파일을 추가하고, 설정 파일에서 keystore 경로 및 비밀번호를 설정합니다.

2. 애플리케이션 실행:

   애플리케이션을 실행하면 HTTPS로 서비스가 제공됩니다.
